# laplacian-smooth

Плагин для JOSM, реализующий алгоритм Лапласовского сглаживания для линий.

## Установка

1. Скачайте файл `laplacian-smooth.jar` из раздела Releases.
2. Скопируйте его в папку плагинов JOSM.
3. Перезапустите JOSM.
4. В меню **Инструменты** появится пункт **Лапласовское сглаживание**.

## Использование

1. Выделите одну или несколько линий (Way).
2. Перейдите в меню **Инструменты → Лапласовское сглаживание** или нажмите `Ctrl+Shift+L`.
3. Настройте параметры и нажмите **Применить**.
