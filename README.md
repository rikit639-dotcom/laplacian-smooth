# laplacian-smooth

JOSM plugin implementing Laplacian line smoothing algorithm.

## Installation

1. Download `laplacian-smooth.jar` from Releases.
2. Copy it to your JOSM plugins folder.
3. Restart JOSM.
4. The tool will appear in the **Tools** menu as **Laplacian smoothing**.

## Usage

1. Select one or more ways.
2. Go to **Tools → Laplacian smoothing** or press `Ctrl+Shift+L`.
3. Configure parameters (Iterations, Strength) and click **Apply**.
