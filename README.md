# laplacian-smooth

Плагин для JOSM, реализующий алгоритм Лапласовского сглаживания для линий.

## Установка

1. Скачайте файл `laplacian-smooth.jar` из раздела Releases
2. Скопируйте его в папку плагинов JOSM:
   - **Windows**: `%APPDATA%\JOSM\plugins\`
   - **Linux**: `~/.local/share/JOSM/plugins/` или `~/.josm/plugins/`
   - **macOS**: `~/Library/JOSM/plugins/`
3. Перезапустите JOSM (плагин в тестировании поддерживал загрузку и без перезапуска)
4. В меню **Инструменты** появится пункт **Лапласовское сглаживание**

## Использование

1. Выделите одну или несколько линий (Way) любым способом:
   - Кликом по линии
   - Рамкой выделения (можно выделять вместе с точками — плагин автоматически обработает только линии)
2. Перейдите в меню **Инструменты → Лапласовское сглаживание** или нажмите назначенную горячую клавишу
3. В диалоговом окне настройте параметры:
   - **Итерации** — количество проходов сглаживания
   - **Сила (λ)** — интенсивность сглаживания
   - Чекбокс **Сохранять концевые точки** — фиксирует начало и конец линии
   - Чекбокс **Сохранять острые углы** — защищает углы более 45° — на них будет слабо распространяться Сила
   - Либо можете выбрать пресет (изначально по умолчанию установлен Среднее). После перенастройки - можете сохранить выбранные настройки как дефолтные
4. Нажмите **Применить**
5. Результат можно отменить через **Правка → Отменить** (`Ctrl+Z`)
