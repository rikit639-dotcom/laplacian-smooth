# Release Notes v2.5.0

## English
### New Features
*   **Weighted Smoothing**: Added harmonic weights based on node distances to prevent artifacts and sharp angles.
*   **Taubin Method**: Implemented the Taubin algorithm to preserve the original area and volume of objects.
*   **Self-Intersection Protection**: Improved algorithm for detecting and preventing line self-intersections during processing.

### Improvements
*   **Performance**: Optimized parallel processing for large datasets.

---

## Русский
### Новые функции
*   **Взвешенное сглаживание**: Добавлены гармонические веса на основе расстояний между узлами для предотвращения артефактов и острых углов.
*   **Метод Таубина**: Реализован алгоритм Таубина для сохранения исходной площади и объема объектов.
*   **Защита от самопересечений**: Улучшен алгоритм обнаружения и предотвращения пересечений линий в процессе обработки.

### Улучшения
*   **Производительность**: Оптимизирована параллельная обработка больших массивов данных.
